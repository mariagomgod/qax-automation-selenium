## Quick Task

###  Instrucciones

1. **Localizando elementos**.

    - Crear un archivo `locators-compra.md` que liste exclusivamente los localizadores necesarios para completar un flujo de compra end-to-end, utilizando la pagina http://www.testingyes.com/onlineshop/
    - Crear un archivo `shadow-iframe-popup.md` que liste los localizadores necesarios para interactuar con elementos dentro de un Shadow DOM, IFrame y PopUp.
        - Shawdow DOM: https://bonigarcia.dev/selenium-webdriver-java/shadow-dom.html
        - IFrame: https://bonigarcia.dev/selenium-webdriver-java/iframes.html
        - PopUp: https://bonigarcia.dev/selenium-webdriver-java/dialog-boxes.html

2. **Acciones avanzadas del clic**:

    - Crear un archivo por cada item, y realice la(s) en la página indicada, usando Playwright.
    - Crear casos de prueb para cad item.

        * Mouse Over: https://bonigarcia.dev/selenium-webdriver-java/mouse-over.html
        * Doble clcik & Click derecho & Select Dropdown : https://bonigarcia.dev/selenium-webdriver-java/dropdown-menu.html
        * Drag & Drop: https://bonigarcia.dev/selenium-webdriver-java/drag-and-drop.html
        * Simular Teclas: https://bonigarcia.dev/selenium-webdriver-java/slow-calculator.html
        * Popup / Alert: https://bonigarcia.dev/selenium-webdriver-java/dialog-boxes.html
        * Manejo de Pestañas: https://demo.automationtesting.in/Windows.html
        * Iframe: https://demo.automationtesting.in/Frames.html

    - Ejecutar todos los casos de prueba.
